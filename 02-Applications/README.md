# Applications Directory

This directory contains active job applications organized by month.

## Structure
- `YYYY-MM/`: Folder for each month.
    - `Company-Name/`: Individual folder per company.
        - `[Name]-CV.md`: The CV version specific to this application.
        - `[Name]-CL.md`: The cover letter or email body.
        - `Application-tracker.md`: Log of dates, contacts, and status.
        - `Research-notes.md`: Notes on the company, role, and interviewers.

## Workflow
1. Create a new company folder.
2. Run research agents to populate `Research-notes.md`.
3. Generate tailored CV and Cover Letter based on research.
4. Update `Application-tracker.md` at every step.
