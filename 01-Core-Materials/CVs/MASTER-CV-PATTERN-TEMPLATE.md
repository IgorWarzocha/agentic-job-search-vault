# MASTER CV PATTERN TEMPLATE

**STRICT ADHERENCE REQUIRED**

## 1. Global Rules
- **Formatting:** Clean Markdown. No `---` lines. No complex tables.
- **Length:** Max 2 pages.
- **Tone:** "Short. Punchy. Action." No "I didn't just X, I did Y". No semicolons.
- **Facts:** Always verify against `CANDIDATE-PROFILE.md`.

## 2. Header
```markdown
# [NAME]
[Phone] | [Email] | [LinkedIn] | [Location]
```

## 3. Profile (Modes)

### MODE A: Technical / Specialist
```markdown
### PROFILE
[Title] who builds [Solution] to solve [Problem]. Focused on [Key Skill] and [Key Skill]. I [Value Proposition]. 
```

### MODE B: Management / Leadership
```markdown
### PROFILE
[Title] with a track record of [Achievement]. I specialize in [Core Competency] and driving [Result]. Experienced in [Area] and [Area].
```
