## Email Application Template

```markdown
Subject: Application for [Role] - [Candidate Name]

Dear [Hiring Manager],

I'm writing to express my strong interest in the [Role] position at [Company].

[Paragraph 2: Why I'm excited about this role]

[Paragraph 3: How my experience aligns with your needs]

[Paragraph 4: What I can bring to the team]

I've attached my CV and cover letter for your review. I'd welcome the opportunity to discuss how my [key strength] and [key strength] could contribute to [Company]'s success.

Thank you for your time and consideration.

Best regards,
[Candidate Name]
[Contact details]

Attachments:
- CV - [Candidate Name].pdf
- Cover Letter - [Candidate Name].pdf
- [Additional materials if applicable]
```

## Application Package Checklist

```markdown
## Application Package: [Company] - [Role]

### Core Materials
- [ ] CV tailored to role
- [ ] Cover letter personalized
- [ ] Professional email format

### Additional Materials (if required)
- [ ] Portfolio samples
- [ ] Case studies
- [ ] Technical assessments
- [ ] References available

### Quality Checks
- [ ] No spelling/grammar errors
- [ ] British English throughout
- [ ] Consistent formatting
- [ ] All requirements addressed
- [ ] Professional tone maintained
- [ ] Files named correctly
- [ ] Follow-up planned

### Submission Details
- [ ] Sent via: [Email/Portal/LinkedIn]
- [ ] Date sent: [YYYY-MM-DD]
- [ ] Confirmation received: [Yes/No]
- [ ] Follow-up scheduled: [Date]
```
