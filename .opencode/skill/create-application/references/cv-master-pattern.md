# Master CV Pattern

<rules>

**Strict adherence required.** The agent MUST NOT invent new sections.

</rules>

<templates>

## Header
```markdown
# [CANDIDATE NAME]
[Phone] | [Email] | [LinkedIn URL] | [Location]
```

## Profile (Select ONE)

**MODE A: Technical / Specialist**
```markdown
### PROFILE
[Title] who builds [Solution] to solve [Problem]. Focused on [Key Skill] and [Key Skill]. I [Value Proposition].
```

**MODE B: Marketing / Growth**
```markdown
### PROFILE
[Title] with a track record of [Achievement]. I specialize in [Core Competency] and driving [Result]. Experienced in [Area] and [Area].
```

**MODE C: Operations / Management**
```markdown
### PROFILE
[Title] focused on process improvement and reliable delivery. I have a track record of [Key Achievement]. I bring structure to chaotic workflows and ensure [Key Outcome].
```

## Signature Project (Optional)
```markdown
### SIGNATURE PROJECT - [PROJECT NAME]
**[Context/Role]**

[Problem statement].
*   **The Problem:** [Description]
*   **The Solution:** [Description]
*   **Impact:** [Quantifiable Result]
*   **Link:** [URL if available]
```

## Experience (Chronological)
```markdown
### RELEVANT EXPERIENCE

**[Job Title]** | [Company], [Location] | [Dates]
*   **[Area of Impact]:** [Action] that resulted in [Quantifiable Result].
*   **[Process Improvement]:** [Action] to solve [Inefficiency].
*   **[Key Deliverable]:** Produced [Output] for [Stakeholder].

**[Job Title]** | [Company], [Location] | [Dates]
*   **[Task]:** [Action] and [Outcome].
*   **[Impact]:** [Action] that [Benefit].
```
